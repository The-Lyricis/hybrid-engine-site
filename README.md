# Hybrid Engine Site

Standalone project site for `Hybrid Engine`.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This repo is set up for Vercel:

- framework: Vite
- output: `dist`
- SPA rewrite: handled by `vercel.json`

Routes currently available:

- `/`
- `/docs`
- `/docs/architecture`
- placeholder docs routes under `/docs/...`
